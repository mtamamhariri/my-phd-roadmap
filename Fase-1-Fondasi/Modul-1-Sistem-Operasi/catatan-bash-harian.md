#Catatan Perintah Bash Harian

#1. 'pwd' (Print Working Directory)
Perintah ini digunakan untuk mencetak direktori (folder) tempat saya berada saat ini.

#2. 'ls' (List)
Perintah ini digunakan untuk menampilkan daftar file dan folder didalam dirtektori saat ini.

#3. 'cd' (Change Directory)
Perintah ini digunakan untuk pindah ke direktori lain. contoh 'cd Modul-Python1' 

#4 'mkdir' (Make Directory)
Perintah ini digunakan untuk membuat folder baru. contoh 'mkdir Modul-Python1'

#5 touch(Create File)
Perintah ini digunakan untuk membuat file baru yang kosong. contoh 'touch catatan-baru.txt'

#6 cp (Copy)
Perintah ini digunakan untuk menyalin (copy) file atau direktori. Anda perlu menentukan sumber dan tujuan. Contoh : cp catatan-baru.txt salinan-catatan.txt

#7 mv (Move)
Perintah ini memiliki dua fungsi:
1.Memindahkan file/direktori ke lokasi lain.
2.Mengganti nama(rename) file/direktori. Contoh (rename): mv catatan-baru.txt catatan-penting.txt contoh (move): mv catatan-penting.txt ../Arsip/

#8 rm (Remove)
Perintah ini digunakan untuk menghapus file. Perhatian : Perintah ini permanen dan tidak memindahkan file ke recycle bin. GUnakan dengan hati-hati. contoh rm salinan-catatn.txt

#9 rmdir (Remove Directory)
Perintah ini digunakan untuk menghapus direktori(folder) yang kosong. contoh rmdir Folder-Lama

cat (Concatenate)
Perintah ini paling sering digunakan untuk menampilkan seluruh isi dari beberapa file teks langsung di terminal contoh cat catatan-penting.txt
